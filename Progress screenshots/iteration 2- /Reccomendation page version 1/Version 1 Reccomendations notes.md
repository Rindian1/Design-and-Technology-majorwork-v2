Currently the reccomendations page feels like just a bunhc of random facts, thrown at the user, and does not fel actionable at all. The two approaches to fix this is 1) organising the reccomendations into completely seperate categories, and 2) making the reccomendations more actionable and specific to the user's energy usage, using the information from the survey. 

Improvements: 
1) Categroising the reccomendations into categroies 
    There should be a seperate opage fir each different tyoe if reccomendation. The reccomendations tab should be gone, and instead there should be multiple tabs for each category of reccomendation. 
        1) General insights 
            These should be general insights based on the current days usage. Things like: "Youve used X% more than this time last week" or "Your highest energy usage was at Y time" etc. 
        2) Appliance specific recommendations 
            These should be recommendations based on the users appliance usage. It should search the web, and offer an appliance with higher energy efficiency, based on the users current appliance. 
                "You are currently spending $X on electricity from this heater. If you instead bought this appliance, you could save $Y per year, considering your current energy spendings" 
            NOTE THAT IT SHOULD ONLY MAKE RECOMMNEDATIONS BASED ON WHAT IT KNOWS. If the user has not provided the specific appliance details, it should not make guesses
        3) Behaviour specific recommendations 
            These should be recommendations based on the users behaviour. For example, if the user has a high energy usage appliance, the recommendation could be to use it during off-peak hours. This can also be things like 
            "You could save $X per year by reducing the temperature of the heater by 3 degrees" 
            "You could save $X per year by turning it off at this time range"  
2) Making the reccomendations more actionable and specific 
    The reccomendations should be more specific and actionable. For example, instead of just saying "You could save $X per year by reducing the temperature of the heater by 3 degrees", it should say "You could save $X per year by reducing the temperature of the heater by 3 degrees, by setting it to 18 degrees instead of 21 degrees"
